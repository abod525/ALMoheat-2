# ALMoheat Accounting System v2.0

نظام محاسبي متكامل لإدارة المخزون والعملاء والفواتير، يدعم نظام الوحدات المزدوجة (عدد ووزن) بشكل دقيق.

## التقنيات المستخدمة

### Backend
- **FastAPI**: إطار عمل بايثون سريع وعصري لبناء الـ APIs.
- **MongoDB**: قاعدة بيانات NoSQL لتخزين البيانات بمرونة.
- **Motor**: محرك MongoDB غير متزامن (Async) للبايثون.
- **Pydantic V2**: للتحقق من صحة البيانات ونمذجتها.

### Frontend
- **React 18**: مكتبة بناء واجهات المستخدم.
- **Tailwind CSS**: لتصميم واجهات احترافية وعصرية.
- **Lucide React**: لمكتبة الأيقونات.
- **Axios**: للتعامل مع طلبات الـ API.
- **React Router DOM**: لإدارة التوجيه داخل التطبيق.

## المميزات الرئيسية
- **نظام الوحدات المزدوجة**: إمكانية بيع المنتجات بالعدد أو بالوزن مع تحديث تلقائي للمخزون في كلا الوحدتين.
- **إدارة العملاء**: تتبع أرصدة العملاء وتاريخ معاملاتهم.
- **الفواتير**: إنشاء فواتير مبيعات مفصلة مع دعم الخصومات والملاحظات.
- **المصروفات**: إدارة المصروفات والإيرادات النقدية.
- **التقارير**: كشوف حسابات العملاء، تقارير المخزون، والتقارير المالية الشاملة.
- **النسخ الاحتياطي**: إمكانية تصدير كافة بيانات النظام إلى ملف Excel.

## طريقة التشغيل

### تشغيل الـ Backend
1. انتقل إلى مجلد `backend`.
2. قم بتثبيت المكتبات: `pip install -r requirements.txt`.
3. تأكد من وجود ملف `.env` يحتوي على `MONGO_URL`.
4. قم بتشغيل الخادم: `python main.py`.

### تشغيل الـ Frontend
1. انتقل إلى مجلد `frontend`.
2. قم بتثبيت المكتبات: `yarn install` أو `npm install`.
3. قم بتشغيل التطبيق: `yarn start` أو `npm start`.
